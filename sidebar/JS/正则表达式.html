# 正则表达式

ps 其实现在都不会自己写正则啦，百度 CV！

# 何为正则？

正则表达式，简单来说就是一套规则 规则是程序员自己定的，用来判断页面的某个内容是否匹配你这个规则

# 使用规则

你要用正则，就需要先 new 一个正则对象 RegExp 正则对象 new RegExp(规则); 调用 test 方法来判断是否符合规则，符合得到 true，不符合得到 false 正则字面量创建： /规则/ 记得//中间要写规则

# 如何使用

// 正则也有字面量创建方法，正则的符号：/规则/ 就是正则符号

var reg = /男/; //等同于var reg = new RegExp('男');
console.log(reg.test('强人锁男')); //true
console.log(reg.test('左右为男')); //true

// 判断内容是否包含女
console.log(/女/.test('2020大家都能找到女朋友')); //true

# 预定义类

预定义类：其实正则里提前规定好的一些特殊符号，用来表达对应的意思

  • . ： 代表判断内容是否有 除了\r，\n 以外
  • \d : 只要有数字（有没有 0
  • \D ： 只要有非数字(除了 0 到 9 以
  • \s ： 不可见字符，你打印的时候看不到，例如：空格，再例如\r \n
  • \S ： 可
  • \w ： 单词字符（ 字母、数字、下划
  • \W ： 非单词字符（找除了字母、数字、下划线以外的）

# 量词

  • "*" 重复零次或更多 x>=0 {0,} //次数要 >= 0
  • "+" 重复一次或更多次 x>=1 {1,} // 至少1次
  • "?" 重复零次或一次 x=(0||1) {0,1}
  • {n} n次 x=n // 必须匹配n次，例{5} ，必须有5次
  • {n,} 重复n次或更多 x>=n // 至少n次，多了不限 例： {5,} 至少5次，多了随便
  • {n,m} 重复出现的次数比n多但比m少 n<=x<=m // n次以上，m次以下 例： {5,8} 至少5次，最多8次

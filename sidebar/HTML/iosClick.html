# ios的哪些兼容。

# 点击，经常我们将事件绑定在元素上，但是逻辑没问题，但是点击事件不触发(亲测试iphone8有这个问题)

解决办法： 在绑定点击的事件上加入元素 cursor: pointer;
这样ios手机才会识别你能够点击，点击事件可正常触发

# 在手机点击经常发现不灵敏，有时候很难触发

解决办法： 将点击的事件用 button 包裹起来，点击事件写在button上面

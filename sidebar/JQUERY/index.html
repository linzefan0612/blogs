# jQuery

jQuery 是一套 JS 库，JS 库其实就是一堆封装好的 JS 代码

# 入口函数

写法：
$(document).ready( function(){} )
简写:
$(function(){  });

# DOM 对象和 jQuery 对象的区别

# DOM 对象

1. 用原生 JS 找到的元素，这就叫 DOM 对象
2. DOM 对象只能用原生的 JS 方法

# jQuery 对象

1. 只有 jQuery 对象才有 jQuery 里的方法
2. $(DOM 对象) 能把 DOM 对象转成 jQuery 对象

# 常用方法

end()   // 回调最初调用的元素
index()   // 获取下标
eq()   // 截取当前下标
next()和nextAll()   // 找到下一个兄弟元素和后面全部兄弟元素
prev()和prveAll()   // 找到上一个兄弟元素和前面全部兄弟元素
sibilings()   // 除了本身，其他兄弟元素
trigger()   // 可以调用别的地方写好的事件
trim()   // 去掉字符串两端的多余的空格
parent()   // 找到父级元素
parents()   // 找到所有父级元素
find()   // 找到子元素，相当于找后代选择器 tr td
children()   // 找到子元素，相当于子代选择器 tr>td
prop()   // 获取和设置  checked,selected,disabled 这种布尔类型的属性

# 类操作

  1. addClass() 添加一个类
  2. removeClass() 删除一个类
  3. hasClass() 判断是否有某个类，有返回 true，没有返回 false
  4. toggleClass() 切换一个类，原来没有这个类就添加，有这个类就删除

# 样式操作

# 获取样式

$(dom).css(样式名);

# 设置样式

设置单个
$(dom).css(样式名,值);

设置多个
$(dom).css({
    样式名 1:值 1,
    样式名 2:值 2,
    });

# 元素宽高

width()和height()   // 就是获取纯粹的宽和高
innerWidth() 和 innerHeight()   // 获取带padding的宽和高
outWidth() 和 outerHeight()   // 获取带padding和边框的宽和高

# 元素位置

offset()
	获得的是盒子与document之间的距离
	有left跟top，这2个值
	类似原生中的 getBoundingClientRect()

position()
	获取的是自身定位盒子的位置
	有left跟top，这2个值

# 创建元素

1. html()
$(dom).html()  // 类似于JS中的innerHTML，包含标签文本
2. $(dom)
创建一个标签，并包装成jQuery对象
类似原生的var 标签=createElement('标签')   标签.innerHTML = '内容
3. append()
添加到元素最后，类似原生的 appendChild
4. prepend()
添加到元素最前
5. before()
添加到某个元素前面，类似原生的 insertBefore
6. after()
添加到某个元素后面
7. appendTo()
自己找到父元素，再添加到末尾
8. prependTo()
自己找到父元素，再添加到最前面

# 节点操作

1. jQuery对象.remove();
会删除自身
2. jQuery对象.empty();
清空子元素，删除事件
3. jQuery对象.html('');
清空子元素，不会删除事件

# 行内属性操作

# 普通类型

attr(属性名)   // 获取行内属性
attr(属性名，属性值)   // 设置行内属性

# 布尔类型

prop()
主要用于checked,selected,disabled等

# 动画

# animate

animate({属性:属性值},速度,回调函数)

参数1：传入一个对象
参数2：动画的持续时间
参数3：动画完成的回调函数

# 滑动

滑动(持续时间，速度，回调函数)，不传为默认

slideDown  下滑
slideUp 上滑

# 淡入淡出

淡入淡出(持续时间，速度，回调函数)，不传为默认

fadeIn 淡入
fadeOut 淡出
fadeToggle 淡出入

# 事件委托

委托可以避免新增元素没有事件的 bug

写法：
jQuery元素.on(事件方法,操作元素,函数)
例：
$('#id').on('click','.div',function(){})

# Ajax

# get

$.get({
	url:接口地址,
	data:请求参数,
	success:回调，接口返回数据
})

# post

$.post({
	url:接口地址,
	data:请求参数,
	success:回调，接口返回数据
})

# ajax

$.ajax({
	type:get 或 post
	url:接口地址,
	data:请求参数,
	success:回调，接口返回数据
})

# 常用参数

url		// 请求地址
data		// 请求参数
success		// 请求数据
dataType		// 手动转换为JS对象
contentType		// 禁止JQ自动创建post表头
processData		// 禁止JQ自动把参数转换成字符串
